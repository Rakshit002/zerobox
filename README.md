# ZEROBOX 🚀

ZEROBOX is a full-stack web application built using modern web technologies.
It provides a clean and interactive interface for managing and previewing emails with secure authentication and a responsive dashboard.

---

## 📌 Features

* 🔐 Authentication and protected routes
* 📧 Email listing interface
* 👁 Email preview functionality
* 📊 Dashboard interface
* 🧭 Sidebar navigation
* ⚡ Fast frontend built with **React + Vite**

---

## 🛠 Tech Stack

**Frontend**

* React
* Vite
* JavaScript
* CSS

**Backend**

* Node.js
* Express.js

## ⚙️ Installation

### 1️⃣ Clone the repository

```
git clone https://github.com/yourusername/zerobox.git
```

---

### 2️⃣ Navigate to project folder

```
cd zerobox
```

---

### 3️⃣ Install frontend dependencies

```
cd frontend
npm install
```

---

### 4️⃣ Run frontend

```
npm run dev
```

---

### 5️⃣ Run backend

```
cd backend
npm install
npm start
```
